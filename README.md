# Dotfiles
- Font: Roboto Condensed Mono Nerd Font
- OSX Keybinds: Karabiner Elements
- OSX Window Manager: Amethyst
- Shell: Zsh (Starship Prompt)
- Terminal: Alacritty
- Text Editor: Neovim
- Theme: Tokyo Night
- Tmux Session Management: [twigsmux](https://github.com/aurokin/twigsmux) (TPM plugin)
